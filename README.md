# DB Schema Synchronizer

Класс для безопасной синхронизации структуры (схемы) боевой базы данных PostgreSQL по образцу тестовой базы. 

Утилита использует `SQLAlchemy` чтения структуры обеих баз данных, вычисляет разницу и генерирует SQL-запросы (`CREATE TABLE`, `ALTER TABLE`) для обновления продакшена. При этом существующие **данные в боевой базе не повреждаются**.

## Структура проекта

* `sync_db.py` — основной Python-скрипт с классом `CustomSchemaSynchronizer`.
* `docker-compose.yml` — конфигурация для локального запуска двух независимых БД (prod_db и test_db).
* `init_prod.sql` — дамп старой структуры боевой базы с тестовыми данными (имитация продакшена).
* `init_test.sql` — дамп новой структуры (имитация локальной разработки с новыми колонками и таблицами).

## Требования

* Docker и Docker Compose (для развертывания тестового стенда)
* Python 3.10+

## Установка

1. Склонируйте или скачайте проект в рабочую директорию.
2. Установите необходимые Python-зависимости:
   ```bash
   pip install -r requirements.txt
  ```
